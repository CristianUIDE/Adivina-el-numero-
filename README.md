# Proyecto Integrador

# Juego Adivina el Número

## Cristian Yaule

## Objetivo del sistema

Desarrollar un programa en Python que permita al computador adivinar el número pensado por el usuario mediante un proceso de búsqueda binaria, aplicando los conocimientos adquiridos durante la asignatura, como estructuras de decisión, estructuras repetitivas, funciones y organización del código.

---

## Descripción del proyecto

El sistema consiste en un juego interactivo en el que el usuario piensa un número entre 1 y 100 sin revelarlo. Posteriormente, el programa realiza diferentes intentos para descubrir dicho número y el usuario únicamente debe indicar si el número mostrado es mayor, menor o correcto.

El programa ajusta automáticamente el rango de búsqueda hasta encontrar la respuesta correcta, reduciendo el número de intentos gracias al uso del algoritmo de búsqueda binaria.

---

## Funcionalidades

- Mostrar un mensaje de bienvenida e instrucciones al usuario.
- Solicitar al usuario que piense un número entre 1 y 100.
- Realizar intentos automáticos para adivinar el número.
- Solicitar la respuesta del usuario mediante las opciones:
  mayor
  menor
  correcto
- Validar que la respuesta ingresada sea correcta.
- Ajustar el rango de búsqueda según la respuesta del usuario.
- Contabilizar la cantidad de intentos realizados.
- Finalizar el juego cuando el número sea encontrado.
- Mostrar un mensaje indicando el número de intentos utilizados.
